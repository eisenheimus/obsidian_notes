
<br>

##### /etc/ssh/sshd_config:
```systemd
# Разрешить проброс портов
AllowTcpForwarding
```
<br>

```bash
# Проброс
ssh -L 9000:localhost:8000 server2

# `9000` — локальный порт на вашей машине.
# `localhost:8000` — адрес и порт на **удалённом сервере** `server2` (т.е. сервис должен быть запущен на `server2:8000`).
```
<br>

> При подключении к `localhost:9000` на вашей машине трафик будет перенаправлен через SSH-туннель на `server2:8000`.
