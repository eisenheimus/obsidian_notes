**Права задаются как сумма значений. Например:**

- `7` = 4 + 2 + 1 = `rwx` (чтение + запись + выполнение)
- `6` = 4 + 2 = `rw-` (чтение + запись)
- `5` = 4 + 1 = `r-x` (чтение + выполнение)
- `4` = 4 = `r--` (только чтение)
<br>

```bash
sudo chmod 777 /home/$USER/.ssh
```

Значение:

|   Чтение  | Запись    |  Выполнение   |
| --- | --- | --- |
|  4   |  2   |  1   |
|  r   |  w   |  x   |


**Три категории пользователей:**

1. **Владелец** (user/owner) — первая цифра
2. **Группа** (group) — вторая цифра
3. **Остальные** (others) — третья цифра

| Ключ | Значение  |  Расшифровка ключа   | 
| ---- | --------- | --- |
| u    | владелец  |  user   |
| g    | группа    |  group   |
| o    | остальные |  ohers   |
| a    | все       |  all  |

```bash
sudo chmod a+rwx /home/$USER/.ssh

#или
sudo chmod u+rwx,g+rx,g-w,o+r,o-wx myfile
```
<br>

```bash
chmod 700 ~/.ssh                # drwx------ (только владелец)
chmod 600 ~/.ssh/id_rsa         # -rw------- (приватный ключ)
chmod 644 ~/.ssh/id_rsa.pub     # -rw-r--r-- (публичный ключ)
chmod 644 ~/.ssh/authorized_keys # -rw-r--r-- (но лучше 600)
```
<br>

```bash
chmod 755 file    # Владелец: rwx, Группа: r-x, Остальные: r-x
chmod 644 file    # Владелец: rw-, Группа: r--, Остальные: r--
chmod 700 dir     # Только владелец имеет полный доступ
```
<br>

#### Строка прав: `drwxrwxrwx`

- **`d`** = Directory  
- **`r`** = Read 
- **`w`** = Write  
- **`x`** = Execute 

> **`x` (выполнение)** для каталога означает возможность _войти_ в него и получить доступ к файлам внутри (даже если нет права `r`).
> Без `x` вы не сможете использовать `cd` в каталог, даже если есть `r`.

| значение  |   |   |
|---|---|---|
|`-`|Обычный файл|Текстовые файлы, бинарники, скрипты и т.д.|
|`d`|Директория (каталог)|Папка для хранения других файлов|
|`l`|Символическая ссылка (symlink)|Указатель на другой файл/каталог (`→`)|
|`c`|Символьное устройство|Устройства, работающие с потоками байт (например, `/dev/tty`, `/dev/null`)|
|`b`|Блочное устройство|Устройства с поблочным доступом (диски: `/dev/sda`, `/dev/nvme0n1`)|
|`p`|Именованный канал (FIFO)|Межпроцессное взаимодействие (через `mkfifo`)|
|`s`|Сокет|Межпроцессное взаимодействие (сетевые/локальные сокеты)|
