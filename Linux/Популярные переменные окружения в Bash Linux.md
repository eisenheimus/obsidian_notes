|   |   |   |
|---|---|---|
|`$USER`|Имя текущего пользователя|`kristina`|
|`$HOME`|Домашняя директория пользователя|`/home/kristina`|
|`$PATH`|Список директорий для поиска исполняемых файлов|`/usr/bin:/bin:/usr/local/bin`|
|`$PWD`|Текущая рабочая директория|`/home/kristina/projects`|
|`$OLDPWD`|Предыдущая рабочая директория|`/home/kristina`|
|`$SHELL`|Путь к текущей оболочке|`/bin/bash`|
|`$HOSTNAME`|Имя хоста системы|`myserver`|
|`$UID`|ID текущего пользователя|`1000`|
|`$SUDO_USER`|Имя пользователя, вызвавшего sudo|`kristina`|
|`$SUDO_UID`|UID пользователя, вызвавшего sudo|`1000`|
|`$TERM`|Тип терминала|`xterm-256color`|
|`$LANG`|Языковая локаль|`en_US.UTF-8`|
|`$EDITOR`|Предпочитаемый текстовый редактор|`nano` или `vim`|
