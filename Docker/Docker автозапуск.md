Самый простой способ — добавить контейнеру политику перезапуска. Он будет автоматически запускаться при старте Docker (и, следовательно, при загрузке системы)

unless-stopped`: Всегда перезапускать контейнер, кроме случаев, когда он был остановлен вручную командой `docker stop`

```bash
#  добавить политику для вашего контейнера `planka-app`
docker update --restart unless-stopped planka-app
```

```bash
# После этого убедитесь, что сам демон Docker запускается автоматически при старте системы
sudo systemctl enable docker
```
